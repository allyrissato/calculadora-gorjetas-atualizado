# 🧮 Calculadora de Gorjetas

Uma aplicação web simples e intuitiva para calcular gorjetas em restaurantes e bares. Ideal para dividir a conta com amigos de forma prática e justa.

## ✨ Características

- 💰 Calcula gorjeta baseada na qualidade do serviço
- 👥 Divide a gorjeta entre múltiplas pessoas
- 🎨 Interface moderna e responsiva
- 📱 Funciona em qualquer dispositivo (desktop, tablet, mobile)
- ✅ Validação de entrada de dados
- 🚀 Rápido e fácil de usar

## 🎯 Como Usar

1. **Insira o valor da conta** (em Reais)
2. **Selecione a qualidade do serviço**:
   - 30% - Incrível
   - 20% - Bom
   - 15% - Aceitável
   - 10% - Ruim
   - 5% - Péssimo
3. **Defina quantas pessoas** vão dividir a conta
4. Clique em **Calcular** para ver o resultado

## 📝 Exemplo de Uso

**Cenário:** Você está em um restaurante com 2 amigos, a conta ficou R$ 150,00 e o serviço foi bom (20%).

1. Insira: **150** (valor da conta)
2. Selecione: **20% - Bom** (qualidade do serviço)
3. Digite: **3** (número de pessoas)
4. Clique: **Calcular**

**Resultado:** Cada pessoa pagará **R$ 10,00** de gorjeta
